A lightweight Web Component for displaying code with syntax highlighting, copy functionality, line numbers, and theme support.

function greet(name) { return `Hello, ${name}!`; }
View Demos API Documentation

Features

Syntax Highlighting

13+ languages supported via highlight.js including JavaScript, Python, CSS, HTML, JSON, YAML, and more.

Multi-File Tabs

Group related files in a tabbed interface with <code-block-group>.

Line Highlighting

Draw attention to specific lines with the highlight-lines attribute.

Download Code

One-click download to save code snippets as files.

Light & Dark Themes

GitHub-inspired color schemes with full CSS custom property support.

Fully Customizable

15+ CSS custom properties for complete visual customization.

Quick Start

Installation

Install via npm:

npm install @profpowell/code-block

Or include via CDN:

<script type="module" src="https://unpkg.com/@profpowell/code-block"></script>

Basic Usage

<code-block language="javascript"> const greeting = "Hello, World!"; console.log(greeting); </code-block>
