<code-block>

A lightweight Web Component for displaying code with syntax highlighting, copy functionality, line numbers, and theme support.

View Demos API Documentation

Features

Syntax Highlighting

13+ languages supported via highlight.js including JavaScript, CSS, HTML, JSON, YAML, PHP, and more.

Multi-File Tabs

Group related files in a tabbed interface with <code-block-group>.

Line Highlighting

Draw attention to specific lines with the highlight-lines attribute.

Share & Download

One-click download or open in CodePen with built-in share functionality.

Light & Dark Themes

GitHub-inspired color schemes with full CSS custom property support.

Lazy Loading

Defer syntax highlighting until visible for optimal page performance.

Quick Start

Installation

Install via npm:

npm install @profpowell/code-block

Or include via CDN:

<script type="module" src="https://unpkg.com/@profpowell/code-block"></script>

Basic Usage

Import and use the component:

import '@profpowell/code-block' <code-block language="javascript"> const greeting = "Hello, World!"; console.log(greeting); </code-block>

Result

const greeting = "Hello, World!"; console.log(greeting);
